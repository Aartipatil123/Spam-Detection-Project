# Spam Detection System (Python & Machine Learning)

## Project Overview
This project is a **Spam Detection System** developed using **Python, Machine Learning, and Tkinter GUI**.  
It classifies messages or reviews as **Spam** or **Not Spam** using a trained ML model.

The system provides:
- User authentication
- GUI-based spam detection
- Pre-trained machine learning model
- SQLite database support

---

## Features
- User Login & Registration  
- Machine Learning based Spam Detection  
- GUI using Tkinter  
- Text preprocessing & vectorization  
- SQLite database integration  
- Clean and structured project layout  

---

## Technologies Used
- **Language:** Python  
- **GUI:** Tkinter  
- **Machine Learning:** Scikit-learn  
- **Model:** Logistic Regression / SVM  
- **Database:** SQLite  
- **Libraries:** pandas, numpy, joblib, pickle, pillow
